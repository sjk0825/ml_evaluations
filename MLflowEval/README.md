
### Set environment
```
$ conda create -n mlflow_env python==3.10
$ conda activate mlflow_env
$ (mlflow_env) cd MLflowEval
$ (mlflow_env) pip install -r requirements.txt
```

### Run evaluation
```
$ (mlflow_env) cd MLflowEval
$ (mlflow_env) python evaluation.py

```
