### Set environment
```
$ conda create -n deepeval_env python==3.10
$ conda activate deepeval_env
$ (deepeval_env) cd DeepEval
$ (deepeval_env) pip install -r requirements.txt
```

### Run evaluation
```
$ (deepeval_env) cd DeepEval
$ (deepeval_env) python evaluation.py

```
